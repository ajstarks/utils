# bezpoints -- generate decksh markup for a filled quadratic bezier curve

```
sage of bezpoints:
  -color string
    	color (default "gray")
  -cx float
    	control x (default 50)
  -cy float
    	control y (default 80)
  -ex float
    	end x (default 80)
  -ey float
    	end y (default 50)
  -n int
    	number of points (default 100)
  -opacity float
    	opacity (default 50)
  -sx float
    	start x (default 20)
  -sy float
    	start y (default 50)
```
