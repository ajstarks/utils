# fox -- inspired by "Fox I" by Anni Albers (using deck markup)

![hot](hot.png)

```fox -color 0:40 -shadow 60 -bgcolor black```

![example](fox.png)

```fox -color ajstarks -w 20,80,2.5 -h 20,80,2.5  -shadow=30 -bgcolor linen```
